# Top 10 Frontend UI/UX Design & Development Skills
*Ranked by up-to-date standards, practical usefulness, and uniqueness (duplicates removed)*

---

## 1. frontend-development
Core production guidelines for React/TypeScript application development, covering modern patterns including Suspense, lazy loading, `useSuspenseQuery`, MUI v7 styling, TanStack Router configuration, type-safe data fetching, and performance optimization for production-ready apps.

## 2. nextjs16-tailwind4
End-to-end workflow for building luxury-grade Next.js 16 applications with Tailwind CSS v4, integrating Radix UI (shadcn) components and Framer Motion animations. Includes CSS-first theming, avant-garde UI design, security audits, and performance tuning for high-end web experiences.

## 3. tailwind-patterns
Specialized reference for Tailwind CSS v4, covering CSS-first configuration, container queries, modern utility patterns, and design token architecture to build scalable, consistent styling systems without deprecated v3 workflows.

## 4. ui-styling
Toolkit for creating accessible, visually consistent UIs using shadcn/ui (built on Radix UI + Tailwind CSS). Includes prebuilt accessible components (dialogs, forms, tables), theme customization, dark mode implementation, responsive layout design, and canvas-based visual asset creation.

## 5. nextjs-react-expert
Vercel Engineering-backed performance optimization guides for React and Next.js applications, focusing on eliminating render waterfalls, reducing bundle size, and implementing efficient server/client-side rendering and data fetching strategies.

## 6. web-design-guidelines
Audit framework for reviewing UI code against Web Interface Guidelines and WCAG accessibility standards. Used to validate UX compliance, catch accessibility gaps, and ensure adherence to frontend best practices before release.

## 7. web-frameworks
Full-stack web app development guides for Next.js (App Router, Server Components, SSR/SSG/ISR), Turborepo monorepo management with task pipelines and remote caching, and RemixIcon integration for 3100+ SVG icons in outlined/filled styles.

## 8. aesthetic
Comprehensive design system framework covering four stages: BEAUTIFUL (aesthetic principles), RIGHT (functionality/accessibility), SATISFYING (micro-interactions), and PEAK (storytelling). Includes workflows for inspiration analysis via Chrome DevTools, AI-multimodal design iteration, and design documentation.

## 9. avant-garde-design-v4
Anti-generic frontend design skill for building distinctive, production-grade interfaces. Specialized for luxury/premium brand experiences, Tailwind v3 to v4 migrations, mobile navigation debugging, and landing page creation that avoids generic AI design tropes.

## 10. frontend-design
Principle-first design thinking guide for web UI, covering component design, layout structure, color scheme selection, and typography choices. Focused on teaching reusable decision-making frameworks rather than fixed design values.

| Skill Name | Reason for Exclusion |

| `frontend-design-basic` | Narrow overlap with `avant-garde-design-v4` for generic distinctive interface generation |
